## Saavn Labs App

A full-featured Expo/React Native app built with the `@saavn-labs/sdk` to browse and play music from JioSaavn.

### Prerequisites

- Node.js 18+
- Yarn or npm)

### Setup

1. Install dependencies: `npm install` (or `yarn install`).
2. Start the dev server: `npm run start`.
3. Use the Expo Go app or an emulator to load the project.

### Notes

- The app relies on `@saavn-labs/sdk` for JioSaavn API access and playback orchestration.
- Platform-specific configuration may be required depending on your client setup.
